เป็น project เกมที่เป็นแนว action-rpg ที่ผมกำลังลองทำพัฒนาขึ้นมา
โดยในเกมได้มีการปรับใช้ Observer Pattern สำหรับระบบเก็บ energy โดยผ่าน EventManager
นอกจากนี้ก็มีการใช้เรียกการใช้ Apply Speed buff ผ่าน Observer pattern ด้วยเช่นกัน
อีก pattern นึงที่ใช้ก็จะเป็น Command pattern ที่ใช้สำหรับ Activate buff ต่างๆให้แก่ตัวละคร
เช่น Speed buff, Attack buff, Energy buff
สาเหตุที่เลือกใช้ pattern พวกนี้ีเพราะทำให้การ develop เกมในระยาวมีความสะดวกและเป็นระบบมากขึ้น
เพราะเข้ากับหลักการ open for adding but close for editing ทำให้การเพิ่มระบบต่างๆง่ายมากขึ้น
เพราะสามารถเพิ่มระบบได้ โดยไม่ต้องแก้ไข code เดิม
ระบบ Command pattern ทำให้สามารถเรียกใช้ Buff ได้โดยผ่านตัวกลางอย่าง CommandManager ทำให้
แต่ละ code ไม่ต้องรู้จักกันให้วุ่นวายเยอะ เพียงรู้จักตัวกลางก็เพียงพอ ทำให้ลดโอกาสการพังของ code ได้ดีขึ้น
